# Advanced Encryption & Decryption Tool

This tool allows users to encrypt and decrypt text using AES-256 encryption.

## Features
- AES-256 Encryption
- Random key generation
- Clean HTML/CSS frontend
- Python (Flask) backend

## How to Run
1. Install dependencies:
   pip install flask cryptography

2. Start backend:
   python backend.py

3. Open frontend:
   frontend/index.html
